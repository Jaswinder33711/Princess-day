Princess Day — Pink Theme
Files:
 - index.html  -> Single-file website. Open in browser to use.
How to use:
 - Open index.html in any modern browser.
 - Use 'कपल फोटो जोड़ें' to upload a photo (optional).
 - Click 'QR बनाएँ' to generate a QR that encodes the page URL with your message/photo.
 - Use 'प्यार के साथ भेजें' to copy the shareable link.
Customization:
 - To change default message/name, edit the HTML (search for default texts).
